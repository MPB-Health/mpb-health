Shared packages & infrastructure

Reusable code powering every app — one library to rule each domain
@mpbhealth/database
Supabase client, generated types, and migration pipeline. The single connection layer for all apps.
@mpbhealth/auth
Shared authentication layer with Supabase Auth, session management, and role-based access.
@mpbhealth/ui
Shared React component library and design system. Consistent look and feel across all surfaces.
@mpbhealth/crm-core
CRM domain logic: leads, pipeline, email, deals, accounts, automation, and Zoho migration.
@mpbhealth/advisor-core
Advisor workflows: training, meetings, content, forms, tickets, chat, events, and portal settings.
@mpbhealth/champion-core
Champion Advisor OS: engagement inbox, sequences, compliance, and AI-powered assistant tools.
@mpbhealth/admin-core
Admin domain logic: builds on advisor-core for CMS, user management, and platform operations.
@mpbhealth/plans-core
Plan data, pricing rules, and rate calculation engine shared across quote and enrollment flows.
@mpbhealth/config
Shared environment configuration, feature flags, and Tailwind design presets.
@mpbhealth/utils
Shared utilities, Zod validation schemas, sanitization, and date/format helpers.
Supabase Edge Functions
30+ serverless functions: bulletin notifications, password resets, bulk advisor import, ticket proxy, chat service, portal SSO, CRM sync agents, email delivery, and more. The shared API layer all apps call into.
